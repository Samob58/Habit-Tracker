# Habit Tracker

A simple daily habit counter — track anything you want to count each day. Data resets daily but history is saved. Includes analytics with streaks, charts, and stats.

## Features
- Add/remove/edit counters
- Daily reset (history preserved)
- Analytics tab with charts and streaks
- Works offline (PWA)
- Installable on phone home screen

## How to use on your phone
1. Open the GitHub Pages URL in **Safari** (iPhone) or **Chrome** (Android)
2. **iPhone:** Tap the Share button → "Add to Home Screen"
3. **Android:** Tap the three-dot menu → "Add to Home Screen"

It will appear as an app icon and open full screen.

## Files
- `index.html` — main page
- `style.css` — styling
- `app.js` — all logic + data tab
- `manifest.json` — PWA config (makes it installable)
- `sw.js` — service worker (offline support)
